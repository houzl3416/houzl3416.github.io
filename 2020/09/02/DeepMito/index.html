DeepMito

DeepMito：利用卷积神经网络精确预测蛋白质亚线粒体定位

Abstract

Motivation

蛋白质在细胞室中的定位问题会影响其功能的正确发挥

Result

通过卷积神经网络和其他深度学习方法，来预测蛋白质亚线粒体细胞定位（亚线粒体细胞出现在基质，外膜，内膜，膜间区的不同位置）

在一个包含424个线粒体蛋白的数据集上进行了交叉验证训练和测试。最后在一个高度精选的人类线粒体蛋白质数据集上进行验证。

Availability and implementation

数据集：http://busca.biocomp.unibo.it/deepmito

GitHub：https://github.com/BolognaBiocomp/deepmito

Materials and methods

Datasets

The SM424-18 dataset

  • 十折交叉验证（BLASTP聚类之后分割子集）

The SubMitoPred dataset

http://proteininformatics.org/mkumar/submitopred/download.html

  • 五折交叉验证
  • 两种交叉验证分裂方式（随机分裂和使用BLASTP在序列聚类之后再随即分割）

The Human Cell Atlas dataset

为了评估DeepMito进行基因组规模分析的能力，在这里采用了从人类蛋白质图谱项目的细胞图谱部分提取的数据集

http://busca.biocomp.unibo.it/deepmito

Feature descriptors

采用三种特征类型

  • Residue one-hot encoding (SEQ)
  • Residue physical–chemical properties (PROP)
  • Position Specific Scoring Matrices (PSSM)

Result

Assessing the contribution of the different features

五次在SM424-18数据集上的实验，分别为：

  • SEQ
  • PROP
  • PSSM
  • SEQ+PROP
  • PSSM+PROP

试验结果表明，单独考虑序列和蛋白质性能时，PSSM有不错的预测性能，但是党PSSM和PROP结合后，预测能力进一步提升。将PSSM+PROP的结果作为DeepMito在该数据集上的结果。

Analyzing DeepMito predictions on the SM424-18 dataset

比较DeepMito在动物、植物、真菌蛋白质这些子集上的性能得分。

线粒体膜蛋白定位的预测结果（SP蛋白，MP蛋白，PM蛋白）

Comparing DeepMito with other approaches

进行了两次实验：

用DeepMito在SubMitoPred数据集上进行训练和测试。

  • 和SubMitoPred方法一样采取五折交叉验证
  • 更严格的十折交叉验证

Scoring DeepMito on genomic-scale analysis（基因组规模分析中的评分）

包含一次实验：

使用完整的SM424-18数据集对DeepMito进行了训练，并对Mito-CA-Full数据集中包含的所有1050种蛋白质进行预测。
